# Images

Drop site photos here. Filenames must match what the code expects.

- `hero.jpg` — full-width hero background (≈1920×1080, < 400 KB, ideally WebP/JPG)
- `og-image.jpg` — social share preview (1200×630)
- `menu/` — one photo per menu item, named to match `image:` in `src/content/menu.ts`
  (e.g. `menu/biscoff-heaven.jpg`)
- `gallery/1.jpg … 6.jpg` — gallery grid photos (square, ~1000×1000)

Tips for fast loading:
- Export at the size shown above — don't upload 5000px originals.
- Prefer `.webp` or compressed `.jpg`.
- Keep each image under ~400 KB.
